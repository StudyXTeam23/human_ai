===============================================
          IDE 调试使用说明
===============================================

❌ 错误方式:
   运行: app/main.py
   结果: ModuleNotFoundError

✅ 正确方式:
   运行: run_debug.py
   结果: 服务器正常启动

===============================================

📝 步骤:

1. 在 IDE (PyCharm/IntelliJ) 中打开:
   /Users/yuyuan/studyx_human/web/backend/run_debug.py

2. 右键文件,选择:
   - Run 'run_debug' (运行)
   - 或 Debug 'run_debug' (调试)

3. 访问:
   http://localhost:8000/docs

===============================================

💡 为什么不能直接运行 main.py?

main.py 使用相对导入:
  from app.config import settings
  from app.api import humanize, ...

直接运行时,Python 找不到 'app' 模块。

run_debug.py 会自动设置好所有路径!

===============================================

🔧 可选: 设置 Python 解释器

如果想直接运行 main.py,需要:

1. Settings → Project → Python Interpreter
2. 选择: 
   /Users/yuyuan/studyx_human/web/backend/venv/bin/python

3. 但仍然推荐使用 run_debug.py!
